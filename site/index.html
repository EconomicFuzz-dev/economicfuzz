How It Works
01

Write Attack Scenario

Define attack steps in YAML — manipulation type, target parameters, invariant checks. Templates included for common DeFi exploits.

02

Execute Simulation

Run ecofuzz attack scenario.yaml — each step simulates the economic action, tracks profit, and checks protocol invariants.

03

Evolve with Genetic Fuzzer

Run ecofuzz fuzz scenario.yaml — the genetic algorithm evolves attack parameters over 100 generations to find maximum-damage configurations.

04

Generate Report

Run ecofuzz report — get a severity-rated vulnerability report with estimated losses and attack paths. Export as JSON.
